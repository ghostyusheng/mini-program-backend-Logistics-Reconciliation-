`Logistics Reconciliation Mini-Program (Full-Stack, Production Deployment)`

 `Tech: Python (FastAPI), PostgreSQL, Docker, Taro (React-based), Nginx`
 
Designed and implemented an end-to-end logistics reconciliation system for handling packing lists, invoices, and shipment data.

Built RESTful backend APIs with PostgreSQL schema design and JSONB-based item storage.

Developed cross-platform frontend using Taro, supporting structured data entry and image upload workflows.

Containerised deployment using Docker and reverse proxy with Nginx.

Delivered from requirement analysis to production-ready deployment within tight time constraints.

`Frontend (Taro)->
FastAPI->
PostgreSQL
->
Docker + Nginx`
